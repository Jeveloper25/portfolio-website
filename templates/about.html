About Me

An aspiring Computer Systems Engineer

Computer Systems Engineer

I am a current university student studying Computer Systems Engineering. Tech has always really interested me. Since I was young, I've been fascinated by technologies ranging from video games to embedded systems and robotics, which led me to pursue a career as an engineer so that I may create some of these technologies myself.

Through my exploration into engineering, the software discipline emerged as the most important to learn and the most fun to engage with. It is for that reason that I chose to study Computer Systems Engineering. A benefit of going down the software route is that it allows a single individual to create such a wide breadth of technologies using nothing more than their own computer.

This brings us to this website. The purpose of this website is to host and document my various projects as I create them. For the moment, it documents only a very small number of projects, but as time goes on, I believe it will become a great reflection of my journey as a Computer Systems Engineer. The website itself is also one such project, using a pre-made HTML/CSS template and a Flask server. It will continue to be updated and refined as I create new projects.
