password_security.py

File defining the Password class. Once a Password object has been initialized, the desired password can be passed into the .config method which will call various other methods to analyze the password and get a final score, stored in the .security_score parameter. This .security_score can be evaluated into plain-english using the .get_password_strength method which will return the evaluation, as seen in the project demo.

password_breach.py

File defining functions for retrieving information from pwnedpasswords API. Primary function is pwn_api_check which uses the other two functions, get_password_leaks_count and request_api_data to retrieve data from the API and determine how many times the given password was breached. pwn_api_check is used by the .analyze_leaks method of the Password class defined in the password_security.py file.
