Description

This is a very simple project that allows a user to assess the security of a password.

The program assesses the security of a given password through a self-created algorithm involving the password's length and amount of unique characters. The algorithm also takes into consideration if the password has been involved in any significant data breaches. It does this through the use of the pwnedpasswords API⇗.

This project was created using Python and works using the Password class. The Password class contains several methods and members, but only two methods are important for any external use. These are the .config and .get_password_strength methods that analyze a given password and return an evaluation of its strength.

A demo as well as the python code are available for further details on the project.

Future Improvements

This project may be functional in its current state, but the algorithm is still relatively poor. It will undergo further development, which will reassess the weighting of the algorithm's parameters and incorporate machine learning to allow for more advanced analysis. Below is a list of some of the planned improvements:

  • Change weighting of various Password parameters when determining password strength
  • Incorporate machine learning to help with the implementation of other features
  • Implement feature to recognise common words or patterns within the password
  • Improve demo with enhanced visuals and more in-depth information
  • Create terminal application
