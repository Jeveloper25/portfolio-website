Password Security

Assess the strength of your password

Very simple project that allows the user to assess the strength of a given password. The assessment of the password's security is done through a self-created algorithm involving the password's length and amount of unique characters. The algorithm also takes into account if the password has been involved in any major data breaches. It does this through the use of the pwnedpasswords API⇗.
